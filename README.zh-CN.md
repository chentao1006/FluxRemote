本项目是 [FluxMonitor](https://github.com/chentao1006/FluxMonitor) 的移动设备客户端。

## 简介
Flux Remote 是为 FluxMonitor 提供的官方移动端应用，方便用户在移动设备上远程监控和管理服务。

## 功能
- 实时监控服务状态
- 管理和配置服务
- 多模块支持（如 Docker、Nginx、日志等）
- 多语言界面（中/英）

## 安装与使用
1. 克隆本仓库
2. 使用 Xcode 打开 `ios/Flux Remote.xcodeproj`
3. 运行到你的 iOS 设备或模拟器
